Troubleshooting

GPS Issues

  • • Ensure location services are enabled for RideMate
  • • Try restarting the app if GPS seems inaccurate
  • • Check if you have a clear view of the sky
  • • Verify your phone's GPS is working with other apps
  • • Consider calibrating your phone's compass

App Performance

  • • Close other apps to free up memory
  • • Restart your phone if the app is sluggish
  • • Make sure you have the latest version installed
  • • Check available storage space on your device
  • • Try turning off battery optimization for RideMate

Data & Storage

  • • Back up rides regularly using the export feature
  • • Delete old rides if storage space is limited
  • • Check if iCloud backup includes app data
  • • Contact support if rides disappear unexpectedly
  • • Keep the app updated for the latest data protection

Battery Optimization

  • • Enable power saving mode for longer rides
  • • Reduce screen brightness while riding
  • • Close unnecessary background apps
  • • Use airplane mode in areas with poor signal
  • • Consider a phone mount with charging capability
